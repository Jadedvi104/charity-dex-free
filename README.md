<h2>Installation</h2>

```bash
# Clone the repository
git clone [https://github.com/jadedvi104/repo.git](https://github.com/Jadedvi104/charity-dex-free.git)

# Navigate to the project directory
cd charity-dex-free

# Install dependencies
npm install
or
yarn
```

<h2>Usage</h2>

```bash
# Start the development server
npm run dev
or
yarn dev
```
